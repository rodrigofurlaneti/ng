### Procedimentos - Angular cli

```sh
1 - selecionar o diretório
2 - ng new nome-projeto (Criar projeto)
3 - selecionar o diretório src/app (Raiz)
4 - ng generate component nome-componente (Criar componente)
5 - ng serve
```
